Rodent IV personality creator

Use one of the default options below by pressing one the the buttons like "Ampere" or modify any of the below setting to your like. Once finished click "Save Personality" wich will download your indiviual Character Setting file to your PC. In a final step you have to upload this file again to the Centaur so that you changes added to the configurations

Be aware that you cannot upload any characters found in Rodentiv Character folder since the file format is slightly different!

You can always revert to the default settings on this page by clicking "Reset to default" if you are unsure of your changes.

As you can see, the tool highlights certain options. If you hover a cursor over the highlighted option, a short description of its meaning should appear. Some options are highlited more strongly – this is an indication that I consider them more important for shaping program's style. Of course as a user you are free to disagree and experiment (but please remember that increasing "N likes closed" leads to preferring closed positions).

Explanations of all the options can be found in the section Evaluation parameters. General ideas about personalities creation are described in the document about_personalities.pdf https://github.com/nescitus/Rodent_III/blob/master/docs/about_personalities.pdf

Here you can fine the official Manual to RodentIV https://github.com/nescitus/rodent-iv/blob/master/docs/FinalRodent.pdf
